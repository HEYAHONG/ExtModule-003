# 说明

扩展模块为电路设计的模板，对于基于MCU的模块，主要预定义一些引脚功能，作用与开发板类似。

本扩展模块主要使用芯片:

- [XL2409](https://www.xinlinggo.com/pd.jsp?id=36)

注意:本扩展模块一般不直接使用，一般直接嵌入实际电路设计中
